⚙️ Azure App Configuration

One-time setup at entra.microsoft.com:
1. Entra ID → App registrations → New registration
2. Name it anything (e.g. Vert Price Tool)
3. Supported account types:
  • Work email (@yourco.co.uk) → Single tenant
  • Personal Outlook/Hotmail → Personal accounts only
  • Not sure → Any Entra ID + personal accounts
4. Leave redirect URI blank → Register
5. Go to Authentication → Add a platform → Single-page application
6. Set redirect URI to:
7. API permissions → Add → Microsoft Graph → Mail.Read
8. Copy the Application (client) ID from Overview → paste below
Single tenant → paste your Directory (tenant) ID from Overview.
Personal / any accounts → leave as common.
Your Client ID is saved in your browser only — never sent anywhere except Microsoft's login.
